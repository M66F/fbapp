# fbapp
Fußball Webapp für die Vorlesung Web Entwicklung

Server starten:

Im Directory die Befehle:

     npm install -g grunt-cli (wenn noch nicht gemacht)

     npm install

     grunt

ausführen.

Der Server läuft auf dem Port 3000.
